# Introducción

Este proyecto es una prueba de concepto que implementa una cámara con galería incluida. 
Se ha aplicado el patrón de diseño BLOC para el desarrollo de la aplicación.

# Documentación

La documentación del proyecto puede encontrarse en la carpeta **doc**. 
En esa carpeta podemos encontrar diagramas de clase, priority guides y wireframes de la aplicación.
